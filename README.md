# Segmentasi, Morfologi, dan Kompresi Citra

1. **Set up Virtual Environment**

   ```powershell
   python -m venv venv
   ```

2. **Aktifkan venv**

   ```powershell
   venv/Scripts/activate
   ```

3. **Install dependency**

   ```powershell
   python -m pip install -r requirements.txt
   ```

4. **Jalankan app.py**

   ```powershell
   python app.py
   ```
